Welcome to the HECO 3d Projection Map

We noticed that you do not have a saved Map Layout. You can now position and resize your display as well as hide any elements that you do not want shown.

Each map element is represented by a box and icon here. You can drag an drop them wherever you want to and the cooresponding element that is being projected will move to the cooresponding position

Click on the Settings button of any element and you will see options to resize, hide, or show that element.

When you are finished, click Save Changes to save this layout for the next time you load the application or Cancel Changes if you want to start over.
